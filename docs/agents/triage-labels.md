# Triage Labels

Le skill ragionano in termini di cinque ruoli canonici di triage. Questo file mappa quei ruoli alle stringhe usate nel nostro issue tracker.

| Label in mattpocock/skills | Label nel tracker | Significato                                  |
| -------------------------- | ----------------- | -------------------------------------------- |
| `needs-triage`             | `needs-triage`    | Serve valutazione del maintainer             |
| `needs-info`               | `needs-info`      | In attesa di informazioni dal reporter        |
| `ready-for-agent`          | `ready-for-agent` | Specificata, pronta per un agente AFK        |
| `ready-for-human`          | `ready-for-human` | Richiede implementazione umana               |
| `wontfix`                  | `wontfix`         | Non verrà gestita                            |

Quando una skill menziona un ruolo (es. "applica la label AFK-ready"), usa la stringa corrispondente nella colonna "Label nel tracker".
